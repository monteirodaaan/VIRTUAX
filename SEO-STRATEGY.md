## 3. Core Web Vitals - Otimizações de Performance

### Implementações Necessárias:

1. **Imagens**
   ```typescript
   // Usar Next.js Image com lazy loading
   <Image
     src="/images/banner.jpg"
     alt="Internet fibra óptica de alta velocidade"
     width={1200}
     height={630}
     loading="lazy"
     quality={85}
     placeholder="blur"
   />
   ```

2. **Fonts Optimization**
   ```css
   /* Já implementado com Geist font */
   font-display: swap;
   ```

3. **Preload Critical Resources**
   ```html
   <link rel="preload" as="image" href="/images/logo.webp" />
   <link rel="preconnect" href="https://fonts.googleapis.com" />
   ```

4. **Compression**
   - Habilitar Gzip/Brotli no servidor
   - Minificar CSS/JS (automático com Next.js)
   - Usar WebP para imagens

## 5. Monitoramento e Análise

### Ferramentas Essenciais:

1. **Google Search Console**
   - Verificar propriedade do site
   - Monitorar indexação
   - Identificar erros de rastreamento
   - Acompanhar queries de busca

2. **Google Analytics 4**
   - Configurar eventos personalizados
   - Funis de conversão (formulário de contato)
   - Origem do tráfego
   - Comportamento por cidade

3. **PageSpeed Insights**
   - Monitorar Core Web Vitals
   - LCP (Largest Contentful Paint) < 2.5s
   - FID (First Input Delay) < 100ms
   - CLS (Cumulative Layout Shift) < 0.1

4. **Ahrefs / SEMrush / Ubersuggest**
   - Monitorar backlinks
   - Acompanhar posições no Google
   - Pesquisa de palavras-chave
   - Análise de concorrentes
